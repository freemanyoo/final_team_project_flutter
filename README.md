Java 17, Spring Boot 3.5.4, Flutter SDK 3.9.2, MariaDB, MongoDB, QueryDSL 5.0.0, JWT, Flask 2.3.0, PyTorch 2.0.0, Google Maps API, YouTube API v3, OAuth2 (Google, Naver)
